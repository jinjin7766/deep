## 프로젝트 요약 
YOLO 모델을 활용해 무인점포에서 절도 행위를 탐지하기 위한 딥러닝 기반 개발 프로젝트 입니다. 


## ✅진행 배경 
- **목표 및 방향성 **: 무인 점포 수 증가와 함께 절도범죄 피해가 늘어남에 따라 이를 빠르게 탐지하고 신고가 이루어지게 하기 위한 시스템의 필요성을 느껴 개발을 진행하였습니다.


## 📊성과 및 결과  
- **결과물**: YOLO 기반 탐지 모델 개발 및 테스트, 동영상 파일을 넣고 재생 시키면 절도행위가 검출된 화면이 바로 나오는 UI 화면 개발, 간단한 홈페이지를 만들어 exe 파일로 판매가 이루어질 수 있도록 구현하였습니다.  
- **성과**: 절도 행동 탐지 정확도를 80%대까지 향 

 
## 🛠사용 언어와 라이브러리 
- **프로그래밍 언어**: Python 
- **라이브러리 및 툴**: YOLOv8, OpenCV, TensorFlow, 등


## 📌프로젝트 진행 과정 
- **작업 단계**: 데이터 수집 및 라벨링 -> YOLO 모델 학습 및 튜닝 -> UI 영상 검출 화면 개발 -> 모델, UI화면을 exe 파일로 만듬 -> exe 파일이 판매가 이루어질 수 있는 간단한 웹사이트 개발 
- **역할 분담**: 모델 설계 및 성능 개선, UI화면 개발, exe 파일 만들기 


## ❗️한계점 및 어려웠던 점 
- **제약 조건**: 훈련 데이터 부족, YOLO8모델의 이미지 학습 한계, 제한된 GPU학습 
- **해결 방법**: 데이터 증강 기법 적용
- **아쉬운점**: 컴퓨터 성능 부족, 시간 부

## 💻학습 및 성장 
- **얻은 교훈**: 이미지 학습을 통한 행동을 잡아내는건 한계가 있음, 라벨링의 중요성과 모델 최적화의 복잡성에 대한 깊은 이해
- **기술 향상**: YOLOV8 모델 튜닝 및 OpenCV 활용 능력 향상
- **개인적/팀워크 성장**: 팀원 간의 효율적인 역할 분담과 협업의 중요성을 학습하고 프로젝트 진행 중 발생한 문제에 대한 끊임없는 의사소통의 중요성을 알게 되었습니다. 그리고 개인적으로 문제를 해결하기 위해 계속해서 파고들고 포기하지 않는 마인드가 중요하다는 것을 느꼈습니다.
